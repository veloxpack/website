## Overview

This driver enables Kubernetes pods to mount cloud storage backends as persistent volumes using rclone, supporting 50+ providers (S3, GCS, Azure Blob, Dropbox, SFTP, and more). Plugin name: `rclone.csi.veloxpack.io`.

| master branch | 1.20+ | GA |
| v1.0.0 | 1.20+ | GA |
